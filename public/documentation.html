Movies RESTful API

An API I'm building using node.js, will allow users access information about different movies, directors, and genres.
Also users will be able to sign up, update their personal information and create a list of their favorite movies.

All features of this API:

  • Allow users to see which actors star in which movies
  • Allow users to view information about different actors
  • Allow users to view more information about different movies. (such as the release date and the movie rating)
  • Allow users to create a “Watch” list in addition to their “Favorite Movies” list

This API has the following endpoints:

End point request HTTP method Description
/movies GET Returns a list of All movies in the API
/movies/:Title GET Returns data about a single movie
/movies/:Title/Genre GET Returns data about a genre
/directors/:Name GET Returns data about a director
/documentation GET Serves this documentation file
/users POST Allows new users to be registered. Request body object must contain all fileds: "name", "email", "dateOfBirth")
/users/:Username/FavoriteMovies/:MovieID POST Allows user to add favorite movies.
/users/users/:Username PUT Allows users to update their info
/users/:Username/FavoriteMovies/:MovieID DELETE Allows users to remove a movie from their list of favorites
/users/:Username DELETE Allows a user to delete account
